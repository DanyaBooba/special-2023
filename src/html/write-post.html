У любого пользователя есть возможность получить его опубликованную статью на сайте news.creagoo.ru , если соблюдены правила публикации, о которых дальше пойдет речь.

Стать автором

Вам следует написать статью с названием (заголовком), коротким описанием, основным содержанием (текстом) и изображением.

Формат записи:

  • Название (заголовок) не должен превышать 90-ста символов и обязателен.
  • Короткое описание не должно превышать 42-х символов и не обязательно.
  • Изображение записи должно быть разрешением 1200:600 и соотношением сторон 2:1 соответственно, обязательно к записи.
  • Текст записи может быть любым по размеру, обязателен к записи.

Как опубликовать статью?

Для публикации статьи следует направить приложенный материал к письму на почту: daniil@dybka.ru. Приложенная статья будет рассмотрена, после рассмотрения Вам будет прислан ответ.

Запрещенные темы

На сайте Special запрещено выкладывать материалы с ненормативной лексикой, оскорблениями, разжиганием ненависти. На обсуждение запрещены: темы войны, темы религии, «острые» темы и прочие, которые могут вызвать общих негатив у людей.

Для публикации запрещены рекламные посты без согласования модераторов на их публикацию. Любое наличие рекламных отображений и прочего будет являться причиной для отказа публикации материала.

Все публикуемые посты проходят проверку модераторами. Опубликовать пост без согласия модераторов не предоставляется возможным.

Страница автора

Если Вашу статью одобрили к публикации — для Вас будет создана странциа автора. Вас попросят прислать Вашу фотографию, которая будет приложена к странице автора. Важно: фотография может быть только Вашего лица, другие фотографии не принимаются. По умолчанию будет стоять фотография с Вашими инициалами: имени и фамилии.

Вышеперечисленные критерии не являются обязательными, это факт. В создании сайта есть лишь «рекомендации». Но также существуют и отличия между хорошим сайтом и плохим. Плохим сайтом не хочется пользоваться, тогда какой смысл от этого сайта, если пользователь не хочет получать информацию от сайта?

Активно уделяется внимание пользователю. Существует большое количество людей, которые тестируют отображение сайта. Это действительно важные факторы, которые помогают пользователям дольше оставаться на сайте.
