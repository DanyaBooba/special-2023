О проекте Special

Что такое Special?

Special — это открытый источник медиа от сайта dybka.ru, it.dybka.ru, creagoo.ru и других сайтов, принадлежащих Даниилу Дыбка (dybka.ru).

Главная задача сайта Special — простым языком удобными методами доносить полезную информацию для людей. Когда читать полезную информацию — выбор за Вами, очень удобно во время ожидания пробежаться по одной-второй статье из списка «Интересные». Или целенаправленно просматривать обновления на главной странице.

Стать автором

У Вас есть возможность предложить интересную статью или тему для обсуждения. Для добавления Вас в качестве автора на сайте медиа Special достаточно обратиться на почтовый адрес daniil@dybka.ru.

Ваша заявка будет рассмотрена. В случае успеха поста будет предложено выбрать аватарку для отображения на страницах и Ваш пост будет опубликован на сайте Special.

Запрещенные темы

На сайте Special ведется активная борьба с ненормативной лексикой, оскорблениями, разжиганием ненависти. На обсуждение запрещены: темы войны, темы религии, «острые» темы и прочие, которые могут вызвать общих негатив у людей.

Главная цель медиа Special — донести информацию для людей удобным способом. Сюда не входят задачи разжигания ненависти и оскорбления людей. Все публикуемые посты проходят проверку модераторами. Опубликовать пост без согласия не представляется возможным.

Безопастность сайта

Домен news.dybka.ru использует защищенный метод подключения, используется SSL-сертификат для безопастного и быстрого обмена содержимым с сайтом. Страницы сайта формируются до публикации. Сайт Special не использует базу данных для получения информации, что обеспечивает надежное получение содержимого статей.

Техническая оптимизация

Большое количество внимания было уделено на активную оптимизацию загрузки сайта, доступности содержимого с клавиатуры, адаптивного отображения сайта на разных девайсах, возможности ознакомиться с содержимым при разных условиях.

Вышеперечисленные критерии не являются обязательными, это факт. В создании сайта есть лишь «рекомендации». Но также существуют и отличия между хорошим сайтом и плохим. Плохим сайтом не хочется пользоваться, тогда какой смысл от этого сайта, если пользователь не хочет получать информацию от сайта?

Активно уделяется внимание пользователю. Существует большое количество людей, которые тестируют отображение сайта. Это действительно важные факторы, которые заставляют пользователей оставаться на сайте большее количество времени.

Доступность сайта

Сайт Special доступен для всех пользователей. Имеется возможность перемещаться по элементам с помощью клавиатуры и получать доступ о информации интерактивных элементах. Настроена возможность использования программ для чтения текста с экрана. Прописаны параметры-описания для изображений, чтобы все люди могли узнать, о чем нам говорит размещенная на сайте картинка.
